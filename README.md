# 2. Building Project

Clone the repository:
```
git clone https://github.com/setdebarr/vanguard-2021.git
```

Install required dependencies:
```
cd vanguard-2021
chmod +x install-dependencies.sh
./install-dependencies.sh
```

Make the project:
```
mkdir build && cd build
make ..
```
